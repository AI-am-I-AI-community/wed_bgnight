# Guitar（ギター・音楽活動）

Masayaのギター・音楽活動に関するプロジェクト管理フォルダです。

## 概要
- **活動内容**: ギター演奏、音楽理論学習、作曲、パフォーマンス
- **目的**: 音楽技術の向上、自己表現、楽しみ、コミュニティ参加
- **特徴**: 創造的で表現力豊かな音楽活動
